# BridgePoint MGA — Wholesale Insurance Platform

A Progressive Web App (PWA) for personal auto insurance wholesale brokerage.  
FL ZIP 33545 · Drivers 16–30 · PRD v2.1

**Live demo:** https://vijenderbansal2021.github.io/MGA-Retail-CustomerPortal-Auto-Insurance-Portal/

## Portals
- **MGA / Wholesaler** — Delegated UW · Multi-carrier marketing · Bordereaux · DMS · Compliance
- **Retail Agent** — Submit applications · Receive quotes · Bind · Commission dashboard  
- **Customer** — Self-service quote with TRS telematics scoring
